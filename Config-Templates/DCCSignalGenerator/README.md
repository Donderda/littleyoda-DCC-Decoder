### Beschreibung
Mit dieser Konfiguration kann ein DCC-Signal generiert werden.
Dies kann genutzt werden, um die Vorzüge, wie z.B. Sound, von DCC-Decoder zu
nutzen, aber die Befehle trotzdem per WLAN zu versenden.
